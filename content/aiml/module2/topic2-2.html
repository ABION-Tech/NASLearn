Week 2: Data Preprocessing and Exploratory Data Analysis (EDA)

Understanding Datasets: Types of Data

Data is the foundation of machine learning. It is the raw material that fuels machine learning algorithms. Without understanding its types, quality, and structure, no machine learning model can be effective. The two major types of data include:

1. Numerical Data
  • Continuous: Values that can take any range. For example, age, height, or income.
  • Discrete: Values restricted to specific numbers, often integers. Examples include number of items or shoe size.
2. Categorical Data
  • Nominal: No inherent order. Examples include color or gender.
  • Ordinal: Has a natural order. Examples include education levels or customer satisfaction ratings.

Common Data Formats

Data comes in various formats. Understanding how data is stored and accessed will help you choose the appropriate method for processing and analyzing it:

Format Description Use Case
CSV Comma-Separated Values, used to store tabular data in a text file. Commonly used for storing and transferring datasets.
Excel A spreadsheet application for data manipulation. Used for manual data entry and small-scale analysis.
JSON Lightweight data format, commonly used for structured data exchange in web apps. Best for working with APIs or structured data.
XML A markup language used for storing and transferring data. Common in configurations and document exchange.
Database A structured collection of data stored in a DBMS. Used for large datasets requiring fast access and management.

Data Quality Considerations

Data quality directly impacts the accuracy and reliability of machine learning models. Key issues to address include:

  • Missing Values: Occurs when some data points are incomplete. Techniques like imputation or removal are used to handle this.
  • Outliers: Extreme values that skew data distribution. These can be handled by removal or transformation.
  • Inconsistencies: Mismatches in formats or units. Data needs to be standardized for consistency.
  • Noise: Random variations or errors that obscure real patterns. Filtering or transformation can reduce noise.
