What is HTML?

HTML (HyperText Markup Language) is the standard language for creating webpages. It provides the structure of a webpage using elements enclosed in tags.

Basic Structure of an HTML Document

An HTML document consists of the following structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Welcome to my first webpage.</p>
</body>
</html>

Explanation:

  • <!DOCTYPE html> - Declares the document as an HTML5 document.
  • <html> - The root element that wraps all content.
  • <head> - Contains meta-information, such as character encoding and title.
  • <title> - Sets the title displayed in the browser tab.
  • <body> - Contains the main content of the webpage.

Common HTML Elements

HTML consists of various elements used to structure content. Here are some commonly used tags:

Tag Function Example
<h1> - <h6> Headings (h1 is the largest, h6 is the smallest) <h1>Hello World</h1>
<p> Defines a paragraph <p>This is a paragraph.</p>
<a> Creates a hyperlink <a href="https://example.com">Click Here</a>
<img> Displays an image <img src="image.jpg" alt="Sample Image">
<ul> & <li> Creates an unordered list <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Forms and Input Fields

Forms allow users to submit data to a website. A basic HTML form looks like this:

<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <button type="submit">Submit</button>
</form>

Key Elements:

  • <form> - Defines the form.
  • <input> - Creates input fields (text, email, password, etc.).
  • <button> - Submits the form.

HTML Best Practices

To write clean and efficient HTML, follow these best practices:

  • Use semantic elements like <header>, <nav>, and <footer> instead of <div> where appropriate.
  • Keep code well-structured and indented for readability.
  • Always include the alt attribute in images for accessibility.
  • Use external stylesheets instead of inline styles.
  • Validate your HTML using tools like the W3C HTML Validator.
