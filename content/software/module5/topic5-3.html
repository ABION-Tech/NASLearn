What is CSS?

CSS (Cascading Style Sheets) is used to style HTML content, making webpages visually appealing and user-friendly.

Ways to Apply CSS

CSS can be applied in three main ways:

Method Description Example
Inline CSS Styles applied directly within an HTML element. <p style="color: red;">Hello!</p>
Internal CSS Styles written within a <style> tag in the HTML file. 
<style>
    p { color: blue; }
</style>
External CSS Styles saved in a separate file and linked to the HTML. <link rel="stylesheet" href="styles.css">

CSS Selectors

Selectors define which elements the styles apply to. Here are some common types:

  • Element Selector: Targets all elements of a specific type (e.g., h1).
  • Class Selector: Targets elements with a specific class (e.g., .box).
  • ID Selector: Targets a unique element with an ID (e.g., #main).
  • Group Selector: Styles multiple elements at once (e.g., h1, h2, p { color: red; }).

Basic CSS Properties

Some fundamental properties used in CSS styling:

Property Description Example
color Changes text color. color: red;
background-color Sets background color. background-color: lightgray;
font-size Defines text size. font-size: 20px;
margin Creates space outside an element. margin: 10px;
padding Creates space inside an element. padding: 5px;

CSS Box Model

The CSS Box Model consists of the following layers:

  • Content: The actual text or image inside an element.
  • Padding: Space between content and border.
  • Border: The edge around an element.
  • Margin: Space outside the border.

CSS Best Practices

To maintain clean and efficient CSS, follow these best practices:

  • Use external stylesheets for better maintainability.
  • Keep CSS organized and avoid redundant styles.
  • Use meaningful class names (e.g., .nav-bar instead of .box1).
  • Ensure proper use of the Box Model to avoid layout issues.
  • Test styles on different screen sizes for responsiveness.
