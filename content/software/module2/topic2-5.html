Assignment: Fibonacci Sequence Generator

Objective:

Write a Python program to generate the Fibonacci sequence up to a user-specified number.

Instructions:

1. User Input:

2. Logic:

3. Output:
  • Print the sequence in a clear, comma-separated format.

Example Output:

Enter the maximum number: 20
Fibonacci sequence: 0, 1, 1, 2, 3, 5, 8, 13

Evaluation Criteria:

  • Clean and readable code with proper variable naming and comments.
