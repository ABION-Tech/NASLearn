Topic 3: Control Structures

What Are Control Structures?

Control structures determine the flow of a program's execution. They allow a program to make decisions, repeat actions, and branch into different paths based on specific conditions. Mastering control structures is key to writing dynamic, responsive programs.

Types of Control Structures

  • Conditional Statements – if, else, elif
  • Loops – for loops and while loops
  • Loop Control – break, continue, and pass

1. Conditional Statements

Use conditionals to execute code only when certain conditions are true.

age = 18

if age >= 18:
    print("You're eligible to vote.")
else:
    print("You are not eligible to vote yet.")

elif (short for "else if") lets you check multiple conditions:

score = 75

if score >= 90:
    print("Excellent!")
elif score >= 70:
    print("Good job!")
else:
    print("Keep practicing.")

2. Loops

Loops let you repeat actions.

For Loops

Used to iterate over a sequence (like a list or range).

for i in range(5):
    print("Iteration", i)

While Loops

Used when the number of iterations isn’t known ahead of time.

count = 0
while count < 5:
    print("Count is", count)
    count += 1

3. Loop Control Statements

  • break – Exit the loop immediately.
  • continue – Skip the current iteration and go to the next.
  • pass – Do nothing (used as a placeholder).
for i in range(10):
    if i == 3:
        continue
    if i == 7:
        break
    print(i)

Practice Ideas

  • Write a program that prints “FizzBuzz” for numbers 1–100 (Fizz for multiples of 3, Buzz for multiples of 5, FizzBuzz for both).
  • Create a simple menu-driven program using loops and conditionals.
  • Build a basic number guessing game with a while loop.

Summary

Control structures are the backbone of logical decision-making and automation in software. With if-else statements and loops, you gain the power to make your code smart and efficient. Practice frequently to build strong logic muscles.
