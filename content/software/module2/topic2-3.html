Programming Basics

Topic 3: Functions

1. What Are Functions?

Functions are reusable blocks of code designed to perform specific tasks. They allow developers to write cleaner, more modular, and maintainable code.

Syntax Example:

def greet(name):  # Define the function with 'name' as a parameter
    return f"Hello, {name}!"  # Returns a greeting message

print(greet("Alice"))  # Output: Hello, Alice!
Explanation:
  • def marks the start of the function definition.
  • name is an argument (input) provided when calling the function.
  • return specifies the output of the function.

2. Components of a Function

1. Definition:

Starts with the keyword def followed by the function name and parentheses ().

def function_name():
    # Code block
2. Arguments:

Inputs passed to the function for processing.

def add(a, b):  # a and b are arguments
    return a + b
print(add(3, 5))  # Output: 8
3. Return Statement:

Specifies the value the function sends back to the caller.

def square(num):
    return num * num
print(square(4))  # Output: 16

3. Why Use Functions?

• Avoid Redundancy:

Write the code once and reuse it.

def multiply_by_two(n):
    return n * 2
print(multiply_by_two(5))  # Output: 10
print(multiply_by_two(10))  # Output: 20
• Improved Readability:

Functions with descriptive names make code easier to understand.

def calculate_area(length, width):
    return length * width
• Modularity:

Break programs into smaller, manageable pieces.

Practical Example:

Create a function to calculate the factorial of a number:

def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120

By using functions, programmers can write efficient, modular, and scalable code, simplifying complex problems into manageable tasks.
