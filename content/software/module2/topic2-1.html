This module introduces the fundamental building blocks of programming, focusing on loops, conditionals, and functions. These concepts form the backbone of logical problem-solving in software development.

Topic 1: Conditionals

Conditionals are programming tools that allow decisions to be made based on specific conditions. By using conditionals, programs can execute different pieces of code depending on the situation.

What Are Conditionals?

Conditionals are essential for controlling the flow of a program. They help implement logic by executing certain blocks of code only when a given condition is met.

Example: A basic program to check if a person is eligible to vote. 

if age >= 18:
    print("You can vote.")
else:
    print("You cannot vote.")

Syntax of If-Else

The basic structure of a conditional statement in Python is:


if condition:
    # Code block to execute if condition is True
elif another_condition:  # Optional
    # Code block to execute if the elif condition is True
else:  # Optional
    # Code block to execute if all conditions are False

Explanation:

  • if: Checks the given condition. If it evaluates to True, the associated code block is executed.
  • elif: Checks another condition if the previous one was False. Multiple `elif` blocks can be used.
  • else: Executes the default block of code if all preceding conditions are False.

Best Practices

  • Use proper indentation for readability and to avoid syntax errors.
  • Avoid deep nesting of conditionals. Instead, use logical operators or functions for better readability.
  • Use meaningful variable names to make the code self-explanatory.

Example 1: Grading System

A program to assign grades based on a student's score:


score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")

Example 2: Multiple Conditions

A program to check if a number is positive, negative, or zero:


number = int(input("Enter a number: "))

if number > 0:
    print("The number is positive.")
elif number < 0:
    print("The number is negative.")
else:
    print("The number is zero.")

Practice Problems

  2. Create a program that checks whether a number is divisible by both 3 and 5, divisible by only one of them, or divisible by neither.
  3. Write a program that asks the user for their annual income and calculates the applicable tax based on the following:
    • Income below $10,000: No tax.
