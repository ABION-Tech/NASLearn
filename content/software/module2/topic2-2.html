Topic 2: Variables, Data Types, and Operators

Overview: In this topic, we’ll break down the basic building blocks of programming — variables, data types, and operators. You’ll learn how computers store and manipulate data, and how you can perform operations on that data.

1. Variables: Containers for Data

Variables are like labeled boxes that store information. In Python, you don’t need to declare the type explicitly — Python figures it out for you.

Examples:

name = "Ayo"
age = 23
height = 5.9
is_student = True

Here:

  • name is a string (text).
  • age is an integer.
  • height is a float (decimal).
  • is_student is a boolean (True or False).

2. Data Types: What Can You Store?

Python has several built-in data types:

  • Integers – Whole numbers, e.g., 12, -5, 0
  • Floats – Decimal numbers, e.g., 3.14, -0.5
  • Strings – Text, e.g., "Hello"
  • Booleans – True or False
  • Lists – Ordered collections: [1, 2, 3]
  • Dictionaries – Key-value pairs: {"name": "Ayo"}

3. Operators: Doing Things With Data

Arithmetic Operators:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • // Floor Division (no remainder)
  • % Modulus (remainder)
  • ** Exponentiation (power)

Comparison Operators:

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

Logical Operators:

  • and – Both conditions must be true
  • or – At least one condition must be true
  • not – Reverses the result

4. Type Conversion

You can convert between types using built-in functions:

age = "23"
age = int(age)  # Now it's a number
