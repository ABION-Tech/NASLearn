4.3 CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls the layout, colors, fonts, and overall visual appearance of web pages. It allows you to separate content from presentation, making websites more maintainable and giving you greater control over the design.

CSS Syntax & Selectors

CSS syntax consists of selectors and declaration blocks. Selectors target HTML elements to apply styles, while declaration blocks contain one or more declarations separated by semicolons. Each declaration includes a property and a value, separated by a colon. CSS offers various types of selectors, including element selectors (targeting specific HTML elements), class selectors (targeting elements with a specific class), ID selectors (targeting elements with a specific ID), and more complex selectors like descendant and pseudo-class selectors. Understanding these selectors is crucial for applying styles effectively.

Colors, Typography, and Spacing

CSS provides extensive control over colors, typography, and spacing. Colors can be defined using various formats, including named colors, hexadecimal values, RGB, RGBA, HSL, and HSLA. Typography properties allow you to control font families, sizes, weights, line heights, and text decoration. Spacing properties like margin, padding, and border control the space around and within elements. These properties work together to create visually appealing and readable content, establishing hierarchy and improving the overall user experience.

Box Model

The CSS box model is a fundamental concept that describes how elements are rendered on a web page. Every element is considered a rectangular box with four components: content, padding, border, and margin. The content area contains the actual content, padding surrounds the content, border encloses the padding and content, and margin provides space outside the border. Understanding the box model is essential for controlling layout and spacing. The box-sizing property allows you to change how the box model is calculated, with border-box often being preferred for more intuitive sizing.

Layouts with Flexbox & Grid

Modern CSS provides powerful layout systems: Flexbox for one-dimensional layouts and Grid for two-dimensional layouts. Flexbox is designed for arranging items in a row or column, with properties for controlling alignment, direction, wrapping, and distribution of space. CSS Grid is designed for complex layouts with rows and columns, allowing you to create sophisticated designs with precise control over placement. Both Flexbox and Grid have revolutionized web layout, making it easier to create responsive designs without relying on floats or positioning hacks.

Responsive Design Basics (Media Queries)

Responsive design ensures that websites work well on a variety of devices and screen sizes. Media queries are a key tool in responsive design, allowing you to apply different styles based on device characteristics like screen width, height, orientation, and resolution. By using breakpoints in your CSS, you can create layouts that adapt to different devices, from mobile phones to desktop computers. Responsive design also involves flexible grids, flexible images, and thoughtful content prioritization to ensure a good user experience across all devices.
