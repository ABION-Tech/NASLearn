4.2 HTML (HyperText Markup Language)

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags, telling browsers how to display content. HTML is the foundation of all websites, providing the structure that CSS and JavaScript build upon.

Structure of a Web Page

Every HTML document follows a basic structure. It begins with a doctype declaration, followed by an HTML element that contains a head and a body. The head contains meta-information about the document, such as the title, character encoding, and links to CSS files. The body contains all the visible content of the webpage, including headings, paragraphs, images, links, and other elements. This structure ensures that browsers can correctly interpret and display the content.

Common HTML Elements

HTML provides a variety of elements for different types of content. Headings (h1-h6) create hierarchical structure, paragraphs (p) contain blocks of text, links (a) connect to other pages or resources, and images (img) display visual content. Lists (ul for unordered, ol for ordered) organize items, while div and span elements act as containers for styling purposes. Forms and inputs allow for user interaction, collecting data through text fields, checkboxes, radio buttons, and more.

Semantic HTML and Accessibility

Semantic HTML involves using HTML elements for their intended purpose, providing meaning to the content rather than just presentation. Elements like header, nav, main, article, section, and footer describe their content, making the document more accessible to screen readers and better understood by search engines. Accessibility considerations include proper heading hierarchy, alternative text for images, form labels, and sufficient color contrast. Using semantic HTML improves both accessibility and SEO, making your website more usable for everyone.
