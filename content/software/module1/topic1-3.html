Introduction to JavaScript and TypeScript

Overview

JavaScript and TypeScript are powerful programming languages widely used in web development. JavaScript adds interactivity to web pages, while TypeScript, built on JavaScript, enhances it with optional static typing and modern features.

JavaScript: Adding Interactivity

JavaScript is a versatile scripting language used to add interactivity and dynamic behavior to web pages. It is supported by all modern browsers and works seamlessly with HTML and CSS. Below are key JavaScript concepts:

  • Variables: Store data values. Use var, let, or const.
  • Functions: Define reusable blocks of code.
  • Events: Respond to user actions like clicks or keystrokes.
  • DOM Manipulation: Access and modify HTML elements dynamically.
// Example: JavaScript Basics
let name = "John";
function greet() {
    alert("Hello, " + name);
}
greet();

TypeScript: JavaScript with Superpowers

TypeScript is a superset of JavaScript that introduces static typing and modern features, making it easier to write and maintain complex applications. TypeScript code is transpiled into plain JavaScript to run in the browser.

  • Static Typing: Helps catch errors during development.
  • Interfaces: Define object structures for better code organization.
  • Modules: Support modular code development.
  • Type Annotations: Specify data types for variables and functions.
// Example: TypeScript Basics
let age: number = 25;
function displayAge(userAge: number): void {
    console.log(`Age is ${userAge}`);
}
displayAge(age);

Key Differences Between JavaScript and TypeScript

JavaScript TypeScript
Dynamically typed. Statically typed (optional).
No compilation needed. Requires compilation to JavaScript.
Used for small and medium-scale projects. Preferred for large-scale applications.

Hands-On Example: TypeScript to JavaScript

TypeScript code:

let message: string = "Hello, TypeScript!";
console.log(message);

Transpiled JavaScript code:

var message = "Hello, TypeScript!";
console.log(message);
