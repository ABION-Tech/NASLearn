HTML, CSS, and the Basics of Web Development: A Hands-On Guide

Introduction

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are the fundamental building blocks of the modern web. Together, they provide the structure and style for web pages. This guide will introduce you to the basics of HTML and CSS and provide hands-on examples to help you start creating your own web pages.

HTML: The Building Blocks

HTML defines the structure of a web page. It uses elements enclosed in opening and closing tags. Below are some essential HTML elements:

  • <html>: The root element of an HTML document.
  • <head>: Contains metadata about the page, such as the title and stylesheet links.
  • <title>: Defines the title of the page, shown in the browser tab.
  • <body>: Contains the visible content of the page.
  • <h1> to <h6>: Defines headings of different levels.
  • <p>: Defines a paragraph.
  • <a>: Defines a hyperlink.
  • <img>: Embeds an image.
  • <ul> and <ol>: Define unordered and ordered lists.
  • <li>: Represents a list item.

CSS: Adding Style

CSS is used to style and layout web pages. Below is an example of a basic CSS snippet:

    /* Styling an HTML element */
    body {
        font-family: Arial, sans-serif;
        background-color: #f4f4f4;
    }
    
    h1 {
        color: #333;
    }
    
    a {
        text-decoration: none;
        color: blue;
    }
