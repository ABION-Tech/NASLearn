Part 2: Coding Challenge

Write a Python program that demonstrates OOP principles by implementing a basic system:

Scenario: Library Management System

Create a Python program for a simple library system using OOP. Your program should include the following classes:

  • Book: Attributes - title, author, ISBN.
  • Library: Manages a collection of books (add books, remove books, display books).
  • Member: Attributes - name, member ID, borrowed books.

Requirements:

  • Implement encapsulation by making attributes private and using getter/setter methods.
  • Use inheritance (e.g., SpecialMember class inheriting from Member).
  • Demonstrate polymorphism with method overriding.

Example Expected Output:

Library System:
1. Add Book: "The Great Gatsby" by F. Scott Fitzgerald
2. Register Member: John Doe (ID: 101)
3. Borrow Book: John Doe borrows "The Great Gatsby"
