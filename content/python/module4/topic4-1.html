Basic Data Structures (Review)

In this section, we will review the fundamental data structures: lists, tuples, sets, and dictionaries, focusing on the operations and methods specific to each.

Lists

Operations: Adding, removing, accessing, and slicing elements.

Methods: append(), extend(), pop(), remove(), sort(), etc. 

my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # Outputs: [1, 2, 3, 4]

Tuples

Operations: Indexing, slicing, and using immutable sequences.

Methods: count(), index(). 

my_tuple = (1, 2, 3)
print(my_tuple.index(2))  # Outputs: 1

Sets

Operations: Adding, removing, union, intersection, and difference operations.

Methods: add(), remove(), union(), intersection(). 

my_set = {1, 2, 3}
my_set.add(4)
print(my_set)  # Outputs: {1, 2, 3, 4}

Dictionaries

Operations: Adding, removing, and updating key-value pairs.

Methods: get(), pop(), items(), keys(), values(). 

my_dict = {'a': 1, 'b': 2}
my_dict['c'] = 3
print(my_dict)  # Outputs: {'a': 1, 'b': 2, 'c': 3}
