Course Curriculum

Module 1: Introduction to Python

Get started with Python, understanding its syntax, data types, and writing your first Python program.

Module 2: Variables and Data Types

Learn about variables, data types, and basic operations in Python to handle data efficiently.

Module 3: Data Structures

Study Python's built-in data structures like lists, tuples, sets, and dictionaries, and how to use them for efficient data storage

Module 4: Advanced Data Structures

Study complex data structures such as stacks, queues, linked lists, trees, and graphs. Learn how to implement them in Python and apply them for efficient data organization, searching, and manipulation

Module 5: Functions in Python

Learn how to write and use functions, understanding parameters, return values, and best practices for reusable code.

Module 6: Object-Oriented Programming

Dive into object-oriented programming with Python, understanding classes, objects, inheritance, and polymorphism.

Module 7: File Handling

Learn how to work with files in Python, including reading from and writing to text files, and processing CSV files.

Module 8: Error Handling and Debugging

Understand how to manage errors and exceptions in Python and use debugging techniques to troubleshoot code effectively.

Module 9: Libraries and Frameworks

Explore Python's most popular libraries and frameworks, including NumPy, Pandas, Flask, and Django, for advanced functionality.

Week 10: Capstone Project Execution

Prepare for your internship by focusing on industry expectations and professional development.

Week 11 - 12: Elective Classes

Student shall embark on a two-week journey on any 2 electives availble on our catalogue to broaden their horizons.

Week 13: Internship Preparation

Prepare for your internship by focusing on industry expectations and professional development.

Week 14-23: Internship Proper

Student shall focus on internship aiming to gain industry experience and professional development.

Week 24: Final Completion and Graduation

Student shall finalize all thier works and recieve a certificate on Graduation day, for their hard work in the class session and their dedication during the internship.
