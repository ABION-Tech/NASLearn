**Task 1: Create a Function with Proper Documentation**

Write a Python function that calculates the Body Mass Index (BMI). The function should:

  • Take height (in meters) and weight (in kg) as parameters.
  • Return the BMI value.
  • Include a docstring explaining its purpose, parameters, and return value.
            
def calculate_bmi(weight: float, height: float) -> float:
    """
    Calculate the Body Mass Index (BMI).

    Parameters:
        weight (float): The weight in kilograms.
        height (float): The height in meters.

    Returns:
        float: The BMI value.
    """
    return weight / (height ** 2)

print(calculate_bmi(70, 1.75))  # Example usage

**Task 2: Implement Error Handling**

Modify the calculate_bmi function to handle potential errors, such as:

  • Zero or negative values for height or weight.
  • Invalid data types (e.g., string inputs instead of numbers).
            
def calculate_bmi(weight: float, height: float) -> float:
    """
    Calculate the Body Mass Index (BMI) with error handling.

    Parameters:
        weight (float): The weight in kilograms.
        height (float): The height in meters.

    Returns:
        float or str: The BMI value or an error message.
    """
    try:
        if weight <= 0 or height <= 0:
            return "Error: Weight and height must be positive values."
        return round(weight / (height ** 2), 2)
    except TypeError:
        return "Error: Please provide numerical values."

print(calculate_bmi(70, 1.75))  # Valid input
print(calculate_bmi(-70, 1.75)) # Invalid input
print(calculate_bmi("seventy", 1.75)) # Type error example

**Task 3: Write a Function that Uses a Default Parameter**

Create a function that calculates the final price of a product after applying a discount.

  • The function should take the original price and an optional discount percentage (default is 10%).
  • It should return the final price after the discount is applied.
            
def calculate_discounted_price(price: float, discount: float = 10) -> float:
    """
    Calculate the final price after applying a discount.

    Parameters:
        price (float): The original price of the product.
        discount (float): The discount percentage (default is 10%).

    Returns:
        float: The final price after discount.
    """
    return round(price - (price * (discount / 100)), 2)

print(calculate_discounted_price(100))  # Default discount (10%)
print(calculate_discounted_price(100, 20))  # Custom discount (20%)

Bonus Challenge: Function with Multiple Return Values

Write a function that calculates both the area and perimeter of a rectangle. The function should:

  • Take length and width as parameters.
  • Return both the area and perimeter as a tuple.
            
def rectangle_properties(length: float, width: float) -> tuple:
    """
    Calculate the area and perimeter of a rectangle.

    Parameters:
        length (float): The length of the rectangle.
        width (float): The width of the rectangle.

    Returns:
        tuple: A tuple containing (area, perimeter).
    """
    area = length * width
    perimeter = 2 * (length + width)
    return area, perimeter

area, perimeter = rectangle_properties(5, 3)
print(f"Area: {area}, Perimeter: {perimeter}")
