What are Lambda Functions?

Lambda functions in Python are small, anonymous functions that can have multiple arguments but only one expression.

They are commonly used for short, simple operations without needing a full function definition.

            
# Regular function
def add(x, y):
    return x + y

# Equivalent lambda function
add_lambda = lambda x, y: x + y

print(add(3, 5))  # Output: 8
print(add_lambda(3, 5))  # Output: 8

Using Lambda Functions with Map, Filter, and Reduce

Lambda functions are often used with map(), filter(), and reduce() for functional programming.

Using map()

The map() function applies a function to each element in an iterable.

            
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Output: [1, 4, 9, 16, 25]

Using filter()

The filter() function filters elements based on a condition.

            
numbers = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # Output: [2, 4, 6]

Using reduce() (from functools)

The reduce() function performs a cumulative operation on a list.

            
from functools import reduce
numbers = [1, 2, 3, 4, 5]
sum_result = reduce(lambda x, y: x + y, numbers)
print(sum_result)  # Output: 15

Understanding Variable Scope in Python

Scope determines where a variable can be accessed in a program. There are four types of scopes in Python:

  • Local Scope: Variables defined inside a function.
  • Enclosing Scope: Variables in an outer function, used in nested functions.
  • Global Scope: Variables defined at the top level of a script.
  • Built-in Scope: Variables from Python's built-in functions.

Example of Variable Scope

Local Scope

A variable inside a function is not accessible outside of it.

            
def example():
    x = 10  # Local variable
    print(x)

example()
# print(x)  # Error: x is not defined

Global Scope

A variable declared outside a function is globally accessible.

            
y = 20  # Global variable

def show():
    print(y)

show()  # Output: 20

Modifying Global Variables Inside Functions

Use the global keyword to modify a global variable inside a function.

            
z = 5

def modify_global():
    global z
    z = 10

modify_global()
print(z)  # Output: 10

Enclosing Scope (Nested Functions)

A variable in an outer function can be accessed by an inner function.

            
def outer():
    message = "Hello"

    def inner():
        print(message)  # Accessing outer function's variable

    inner()

outer()  # Output: Hello
