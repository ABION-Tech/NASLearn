Page 2: Defining and Calling Functions

How to Define a Function

A function in Python is defined using the def keyword, followed by the function name and parentheses.

Here is the basic syntax:

            
def function_name():
    # Code block
    print("Hello from the function!")

To execute the function, you need to call it using its name:

            
function_name()  # This will print: Hello from the function!

Calling a Function with Parameters

Functions can accept inputs, known as parameters, inside the parentheses.

Example:

            
def greet(name):
    print("Hello, " + name + "!")

greet("Ayo")  # Output: Hello, Ayo!
greet("Kunle")  # Output: Hello, Kunle!

Using Default Parameter Values

If a parameter is not provided when calling the function, Python uses a default value.

Example:

            
def greet(name="Guest"):
    print("Hello, " + name + "!")

greet()  # Output: Hello, Guest!
greet("Ayo")  # Output: Hello, Ayo!

Returning Values from a Function

Functions can return values using the return statement.

Example:

            
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8

Understanding Variable Scope

Variables inside a function have **local scope**, meaning they are only accessible within that function.

Example:

            
def test_scope():
    x = 10  # Local variable
    print(x)

test_scope()  # Output: 10
print(x)  # Error: x is not defined outside the function
