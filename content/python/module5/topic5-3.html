Page 3: Function Parameters and Return Values

Types of Function Parameters

Functions in Python can have different types of parameters:

  • Positional Parameters: Arguments are matched based on their position.
  • Keyword Parameters: Arguments are matched using the parameter name.
  • Default Parameters: Parameters have predefined values if not provided.
  • Arbitrary Arguments (*args): Allows multiple arguments to be passed.
  • Arbitrary Keyword Arguments (**kwargs): Allows multiple named arguments.

Using Positional Parameters

In a function call, arguments are assigned to parameters in the order they appear.

            
def full_name(first_name, last_name):
    print(first_name + " " + last_name)

full_name("Ayo", "Abion")  # Output: Ayo Abion

Using Keyword Parameters

Keyword parameters allow you to specify arguments by name.

            
def introduce(name, age):
    print(f"My name is {name} and I am {age} years old.")

introduce(age=23, name="Ayo")
# Output: My name is Ayo and I am 23 years old.

Default Parameter Values

If a function is called without an argument, a default value is used.

            
def greet(name="Guest"):
    print(f"Hello, {name}!")

greet()  # Output: Hello, Guest!
greet("Kunle")  # Output: Hello, Kunle!

Using *args for Multiple Arguments

*args allows a function to accept any number of positional arguments.

            
def sum_numbers(*numbers):
    return sum(numbers)

print(sum_numbers(2, 3, 4, 5))  # Output: 14

Using **kwargs for Multiple Named Arguments

**kwargs allows passing multiple named arguments to a function.

            
def user_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

user_info(name="Ayo", age=23, country="Nigeria")
# Output:
# name: Ayo
# age: 23
# country: Nigeria

Returning Multiple Values from a Function

A function can return multiple values as a tuple.

            
def calculate(a, b):
    return a + b, a - b, a * b, a / b

add, sub, mul, div = calculate(10, 5)
print(add, sub, mul, div)
# Output: 15 5 50 2.0
