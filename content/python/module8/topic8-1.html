Stacks

Stacks

A stack is a linear data structure that follows the LIFO principle (Last In, First Out). Operations: push, pop, peek, is_empty.

class Stack:
    def __init__(self):
        self.stack = []
    def push(self, item):
        self.stack.append(item)
Queues

Queues

A queue is a linear data structure that follows the FIFO principle (First In, First Out). Operations: enqueue, dequeue, peek, is_empty.

class Queue:
    def __init__(self):
        self.queue = []
    def enqueue(self, item):
        self.queue.append(item)

Linked Lists

A linked list is a linear data structure where elements, known as nodes, are linked using pointers. Operations: insert, delete, traverse.

        self.head = None

Trees

A tree is a hierarchical structure made up of nodes connected by edges. Binary Tree: max two children per node. Binary Search Tree: left < parent < right.

class BinarySearchTree:
    def __init__(self):
        self.root = None
    def insert(self, data):
        if not self.root:
            self.root = Node(data)
