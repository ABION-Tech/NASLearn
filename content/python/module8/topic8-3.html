Algorithm Complexity and Big O Notation

  • Big O Notation: Describes the time complexity or space complexity of an algorithm as the input size grows. Key complexities include:
    • O(1): Constant time.
    • O(log n): Logarithmic time (binary search).
    • O(n): Linear time (linear search).
    • O(n log n): Log-linear time (merge sort).
    • O(nÂ²): Quadratic time (bubble sort, worst-case quick sort).
  • Analyzing Algorithm Efficiency:
    • Measure the time complexity by counting the number of basic operations performed as a function of the input size.
    • Space complexity refers to the amount of memory an algorithm needs.
