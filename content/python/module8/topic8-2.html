Common Algorithms

Sorting Algorithms

  • Quick Sort:

    Description: A divide-and-conquer algorithm that selects a pivot and partitions the array into two halves: one with elements smaller than the pivot and one with larger elements. Recursively sorts the halves.

    Big O Complexity: Worst-case: O(nÂ²), Average-case: O(n log n).

  • Merge Sort:

    Description: A divide-and-conquer algorithm that divides the array into two halves, recursively sorts them, and merges them back together.

    Big O Complexity: O(n log n) in all cases.

Searching Algorithms

  • Binary Search:

    Description: A search algorithm that works on sorted arrays by repeatedly dividing the search space in half.

    Big O Complexity: O(log n).

  • Linear Search:

    Description: Searches for an element by checking each element in the array one by one.

    Big O Complexity: O(n).
