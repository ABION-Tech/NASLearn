3: Basic Input/Output

Input Function

  • Purpose: Used to receive input from the user during program execution.
  • Syntax:
    • input("Prompt text")
  • Displays the prompt text to the user and returns the user's input as a string.
Example: 
name = input("Enter your name: ")
print(f"Hello, {name}!")

Output Function

  • Purpose: Used to display information to the user.
  • Syntax:
    • print("Message")
  • Outputs the message to the console.
Example: 
print("Welcome to Python Programming!")

Combining Input and Data Conversion

  • Scenario: Input needs to be converted from a string to another data type for processing.
  • Data Conversion Methods:
    • int(): Converts to an integer.
    • float(): Converts to a floating-point number.
Example Code: 
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))
print(f"The sum is {num1 + num2}")

Best Practices for Input/Output

  • Validate User Input: Check for invalid inputs to prevent errors.
    • Example: 
    age = input("Enter your age: ")
if age.isdigit():
    age = int(age)
    print(f"Your age is {age}.")
else:
    print("Please enter a valid number.")
  • Use Formatted Strings: For better readability and clean output.
    • Example: 
    name = input("What is your name? ")
print(f"Nice to meet you, {name}!")

Example Program: Basic Input/Output

# Get user details
name = input("Enter your name: ")
age = int(input("Enter your age: "))

# Output a message
print(f"Hello, {name}! You are {age} years old.")

# Perform a calculation
num1 = float(input("Enter the first number: "))
num2 = float(input("Enter the second number: "))
print(f"The result of adding {num1} and {num2} is {num1 + num2}.")
