Loops in Python

Loops allow you to repeat a block of code multiple times, making it easier to perform repetitive tasks.

`for` Loop:

Used when you know how many times you want to iterate (e.g., over a sequence like a list, string, or range).

Example:

for i in range(5):
    print("This is iteration number", i)

- `range()` function: Generates a sequence of numbers. For example, range(5) generates numbers 0 through 4.

`while` Loop:

Used when you want to repeat a block of code as long as a condition is true.

Example:

counter = 0
while counter < 5:
    print("Counter is:", counter)
    counter += 1
Loop Control Statements:
  1. `break`: Exits the loop prematurely.
  2. `continue`: Skips the rest of the loopâ€™s current iteration and moves to the next iteration.
  3. `else` clause with loops: Executes after the loop completes unless the loop is terminated by break.

Examples:

# break example
for i in range(10):
    if i == 5:
        break
    print(i)

# continue example
for i in range(5):
    if i == 3:
        continue
    print(i)
