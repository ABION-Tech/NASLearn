Primary Data Types in Python

  1. Strings

    Definition: A sequence of characters.

    Example: "Hello, World!"

    Operations:

    • Concatenation: "Hello" + " World" → "Hello World"
    • Slicing: "Python"[0:3] → "Pyt"
  2. Integers

    Definition: Whole numbers without a decimal point.

    Example: 10, -5, 1000

  3. Floats

    Definition: Numbers with decimal points.

    Example: 10.5, -3.14, 0.001

  4. Lists

    Definition: Ordered collection of items, mutable (can be changed).

    Example: [1, 2, 3, 'Python']

    Operations:

    • Adding: my_list.append(4)
    • Removing: my_list.remove(2)
    • Sorting: my_list.sort()
  5. Tuples

    Definition: Ordered collection of items, immutable (cannot be changed).

    Example: (1, 2, 3)

    Usage: Useful for fixed data sets.

  6. Dictionaries

    Definition: A collection of key-value pairs.

    Example:

    {'name': 'Alice', 'age': 25}

    Operations:

    • Add Key-Value Pair: student['grade'] = 'A'
    • Retrieve Value: print(student['name']) → "Alice"

Common Operations on Data Types

  1. Strings:
    • Concatenation: "Python" + " Programming" → "Python Programming"
    • Slicing: "Learning"[0:4] → "Learn"
  2. Lists:
    • Adding Elements:
      • my_list = [1, 2, 3]
my_list.append(4)  # [1, 2, 3, 4]
    • Sorting:
      • my_list = [3, 1, 2]
my_list.sort()  # [1, 2, 3]
  3. Dictionaries:
    • Accessing Values:
      • student = {'name': 'John', 'age': 20}
print(student['name'])  # Output: John
    • Updating Values:
      • student['age'] = 21  # {'name': 'John', 'age': 21}

Example Code: Using Data Types

# Define a dictionary
student = {'name': 'John', 'age': 20}

# Access a value
print(student['name'])  # Output: John

# Modify the dictionary
student['age'] = 21
print(student)  # Output: {'name': 'John', 'age': 21}

# Work with a list
grades = [85, 90, 78]
grades.append(92)
print(grades)  # Output: [85, 90, 78, 92]

Understanding Python's data types is critical for effectively managing and manipulating data in your programs.
