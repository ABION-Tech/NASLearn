Python Programming: Module 2: Variables and Data Types

1: Working with Variables

What Are Variables?

Variables are containers used to store data values, which can later be referenced and manipulated.

length = 5

This statement assigns the value 5 to the variable length.

Naming Conventions

1. Rules for Naming Variables:

  • Must start with a letter (a-z, A-Z) or an underscore (_).
  • Can contain letters, numbers (0-9), and underscores.
  • Cannot use Python keywords like if, for, while, etc.

2. Good Practices:

  • Use descriptive names to make the code readable: customer_name, total_price.
  • Follow consistent styles, such as snake_case: item_count.

Variable Scope

1. Global Variables:

Accessible anywhere in the program.

x = 10  # Global variable

def print_global():
    print(f"Global variable x: {x}")

print_global()  # Outputs: Global variable x: 10

2. Local Variables:

Defined and accessible only within a function or specific block.

def calculate_area():
    length = 5  # Local variable
    breadth = 10
    area = length * breadth
    print(f"The area of the rectangle is {area}")

calculate_area()
# length and breadth are not accessible outside this function.

Example Code: Calculating Area of a Rectangle

length = 5
breadth = 10
area = length * breadth
print(f"The area of the rectangle is {area}")

Output:

The area of the rectangle is 50

Understanding variables and their scope is essential to building functional and organized Python programs.
