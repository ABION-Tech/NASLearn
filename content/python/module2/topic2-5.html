Python Programming: Module 2

Variables and Data Types

Assignment: Average Calculator

Objective

Develop a Python script to calculate and display the average of five user-entered numbers.

Instructions

  1. Input Collection:
    • Use the input() function to prompt the user to enter five numbers.
    • Convert each input to an appropriate data type (float or int).
  2. Calculations:
    • Sum the entered numbers.
    • Divide the sum by 5 to find the average.
  3. Output Display:
    • Print the result in a user-friendly format, explaining what the output represents.

Example Code

# Collecting input from the user
num1 = float(input("Enter number 1: "))
num2 = float(input("Enter number 2: "))
num3 = float(input("Enter number 3: "))
num4 = float(input("Enter number 4: "))
num5 = float(input("Enter number 5: "))

# Calculating the sum and average
total = num1 + num2 + num3 + num4 + num5
average = total / 5

# Displaying the result
print(f"The average of the numbers is {average:.2f}")

Expected Output

Enter number 1: 10
Enter number 2: 20
Enter number 3: 30
Enter number 4: 40
Enter number 5: 50
The average of the numbers is 30.00

Evaluation Criteria

  • Variable and Data Type Usage: Correctly handles numeric input and performs arithmetic operations.
  • Code Readability: Uses meaningful variable names and proper indentation.
  • Accuracy: Ensures correct computation of the average.
  • User-Friendly Output: Provides clear prompts and formatted results.

This assignment reinforces the concepts of variables, data types, and input/output in Python while encouraging good coding practices.
