Data Filtering and Selection

Indexing and Slicing DataFrames

You can filter and select subsets of your data using indexing techniques.

Indexing by Columns
df['Age']  # Access a single column
df[['Name', 'City']]  # Access multiple columns
Indexing by Rows with loc[] and iloc[]

loc[]: Selects data by labels.
iloc[]: Selects data by integer position. 

df.loc[1]  # Access the second row using labels
df.iloc[1]  # Access the second row using integer index
Slicing Rows
df[1:3]  # Access the second and third rows

Filtering Data Based on Conditions

You can filter a DataFrame based on one or more conditions.

Filtering by Condition
df[df['Age'] > 25]  # Select all rows where 'Age' is greater than 25
Multiple Conditions

You can combine conditions using the & (AND) and | (OR) operators.

df[(df['Age'] > 25) & (df['City'] == 'New York')]  # Multiple conditions
