Introduction to pandas

What is pandas?

pandas is a Python library designed for data manipulation and analysis. It provides two key data structures:

  • Series: One-dimensional labeled array.
  • DataFrame: Two-dimensional labeled data structure, like a table of data.

Creating DataFrames and Series

DataFrames are essentially tables, and Series are columns of data within that table.

Creating a Series
import pandas as pd

# Creating a simple Series
s = pd.Series([1, 2, 3, 4], index=['a', 'b', 'c', 'd'])
print(s)
Creating a DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [24, 27, 22, 32],
    'City': ['New York', 'Los Angeles', 'Chicago', 'Houston']
}

df = pd.DataFrame(data)
print(df)
Basic Operations

Here are some basic operations you can perform with pandas:

Accessing data by columns:
df['Name']  # Access the 'Name' column
Accessing data by rows:
df.loc[1]  # Access the second row
Head and Tail:
  • df.head(): Displays the first few rows.
  • df.tail(): Displays the last few rows.
