Iterating Over Data Structures in Python

4. Iterating Over Data Structures

You can iterate over lists, tuples, sets, and dictionaries using loops.

Iterating Over a List:

To loop through the items in a list, you can use a for loop.

# Iterating over a list
for item in my_list:
    print(item)
Iterating Over a Dictionary:

When iterating over a dictionary, you can access both keys and values using the items() method.

# Iterating over a dictionary
for key, value in my_dict.items():
    print(f"{key}: {value}")

Using List Comprehensions:

Python offers a concise way to create lists using list comprehensions, which can also involve iteration:

squares = [x**2 for x in range(10)] # Creates a list of squares
print(squares) # Outputs: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
Conclusion:

Iterating through data structures is a fundamental aspect of programming in Python. It allows for efficient data manipulation and access, making it easier to perform tasks such as filtering, transforming, and processing data.
