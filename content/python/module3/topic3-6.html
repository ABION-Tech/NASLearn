Exercise 1: Student Records Management

Write a program that uses various data structures to store and manipulate a collection of student records. Each student record should include a name, age, and list of grades.

Code:
students = [
    {'name': 'Alice', 'age': 20, 'grades': [85, 90, 92]},
    {'name': 'Bob', 'age': 21, 'grades': [78, 85, 88]},
]

# Accessing student records
for student in students:
    name = student['name']
    age = student['age']
    avg_grade = sum(student['grades']) / len(student['grades'])
    print(f"{name}, Age: {age}, Average Grade: {avg_grade}")
