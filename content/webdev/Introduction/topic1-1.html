Overview

Web development is the process of creating websites and applications for the internet. It consists of two main aspects:

Front-End Development

Focuses on what users see and interact with. It involves using technologies like HTML, CSS, and JavaScript to create responsive and visually appealing interfaces.

Back-End Development

Deals with the logic, databases, and server management that make websites functional. Technologies like Node.js, Express.js, and databases such as MongoDB or SQL manage data and ensure everything works smoothly.

Course Curriculum

Module 1: Introduction to Web Development

Learn the basics of web development, including the difference between front-end and back-end development, and the tools required for both.

Module 2: Intro to CSS and Responsive Web Design

Understand the styling web technologies: CSS for styling content and creating responsive webpages.

Module 3: Responsive Web Design

Learn how to design websites that adapt to different screen sizes and devices using responsive web design techniques.

Module 4: JavaScript Basics

Get introduced to JavaScript, the programming language used to add interactivity and dynamic content to webpages.

Module 5: DOM Manipulation and Event Handling

Learn how to manipulate the Document Object Model (DOM) and handle user events like clicks and keypresses in JavaScript.

Module 6: Introduction to Git and GitHub

Learn how to use version control with Git, and how to collaborate and host your code on GitHub.

Module 7: Front-End Frameworks (Bootstrap)

Learn how to use Bootstrap, a popular front-end framework, to create mobile-first, responsive websites faster.

Module 8: Back-End Development Basics

Get an introduction to back-end development, including working with databases and server-side programming languages like Node.js or PHP.

Module 9: Web Development Tools and Best Practices

Discover tools and techniques to optimize your development workflow, including text editors, debugging tools, and industry best practices.

Week 10: Capstone Project Execution

Prepare for your internship by focusing on industry expectations and professional development.

Week 11 - 12: Elective Classes

Student shall embark on a two-week journey on any 2 electives availble on our catalogue to broaden their horizons.

Week 13: Internship Preparation

Prepare for your internship by focusing on industry expectations and professional development.

Week 14-23: Internship Proper

Student shall focus on internship aiming to gain industry experience and professional development.

Week 24: Final Completion and Graduation

Student shall finalize all thier works and recieve a certificate on Graduation day, for their hard work in the class session and their dedication during the internship.
