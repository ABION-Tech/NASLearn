JavaScript Basics

1.1. What is JavaScript?

JavaScript is a high-level, interpreted programming language that is one of the core technologies of the web, alongside HTML and CSS. It enables you to create dynamically updated content, control multimedia, animate images, and much more.

  • JavaScript vs. Java: Although their names are similar, JavaScript and Java are distinct programming languages with different uses.
  • Client-Side vs. Server-Side: JavaScript is primarily used on the client side, but it can also be used on the server side with environments like Node.js.

Example:

console.log("Hello, World!");

1.2. Embedding JavaScript in HTML

JavaScript can be embedded directly in HTML files or linked as an external script.

Inline JavaScript:

<script>
    alert("Welcome to the website!");
</script>

External JavaScript:

<script src="script.js"></script>

This method is preferred for keeping HTML and JavaScript code separate.
