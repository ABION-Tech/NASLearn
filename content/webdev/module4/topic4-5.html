JavaScript Events

JavaScript events are actions or occurrences that happen in the browser, like clicks, keypresses, and page loads. You can write code that responds to these events, making your web page interactive.

5.1 What are Events?

Events in JavaScript represent user interactions or system-generated occurrences that you can capture and handle with JavaScript.

  • onclick: Triggered when an element is clicked.
  • onmouseover: Triggered when the mouse pointer hovers over an element.
  • onkeyup: Triggered when a key is released after being pressed.

5.2 Event Listeners

Event listeners allow you to define custom behaviors that occur when a specific event is triggered. You can add and remove event listeners dynamically using JavaScript.

1. Adding Event Listeners:

The addEventListener method lets you define what happens when an event occurs on an element. Hereâ€™s an example:

let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("Button was clicked!");
});

2. Removing Event Listeners:

The removeEventListener method lets you stop listening for a specific event on an element. Example:

function handleClick() {
    alert("Button was clicked!");
}

button.addEventListener("click", handleClick);

// Later in the code:
button.removeEventListener("click", handleClick);
