Introduction to JavaScript

1.1 What is JavaScript?

JavaScript is a powerful programming language that enables dynamic behavior on websites. It allows developers to create interactive and responsive web pages by manipulating HTML and CSS, handling user interactions, and communicating with servers.

JavaScript is widely used for:

  • Adding interactivity (e.g., form validation, animations, modals)
  • Creating dynamic web applications
  • Handling asynchronous operations (e.g., fetching data from a server)
  • Building both front-end and back-end applications (Node.js)

1.2 The Role of JavaScript in Web Development

JavaScript plays a crucial role in modern web development as part of the **"triad of web technologies"**:

  • HTML (HyperText Markup Language): Structures the content of a webpage.
  • CSS (Cascading Style Sheets): Styles and enhances the appearance of the webpage.
  • JavaScript: Adds interactivity and dynamic behavior.

Modern frameworks and libraries such as React, Vue.js, and Angular extend JavaScript's capabilities, making web applications more powerful and efficient.

1.3 Setting Up JavaScript

JavaScript can be written directly inside an HTML file or in a separate JavaScript (.js) file. You can include JavaScript in your webpage in three ways:

Inline JavaScript

JavaScript can be written directly within an HTML element using the onclick attribute.

<button onclick="alert('Hello, World!')">Click Me</button>

Internal JavaScript

You can write JavaScript inside the <script> tag within an HTML document.

<script>
        document.write("Hello, JavaScript!");
    </script>

External JavaScript

For better organization and maintainability, JavaScript is often placed in a separate file and linked to the HTML document.

<script src="script.js"></script>

1.4 JavaScript in the Browser

Web browsers come with a built-in **JavaScript Engine** that executes JavaScript code. Popular engines include:

  • V8 Engine (Google Chrome, Node.js): High-performance JavaScript engine used in Chrome and Node.js.
  • SpiderMonkey (Mozilla Firefox): JavaScript engine developed by Mozilla.
  • JavaScriptCore (Safari): Also known as Nitro, used in Apple's Safari browser.

1.5 Writing and Running JavaScript Code

You can test JavaScript code using:

  • The **Browser Console** (press F12 or Ctrl + Shift + I and go to the Console tab).
  • Online editors like **JSFiddle**, **CodePen**, or **JSBin**.
  • A local text editor (e.g., **VS Code**, **Sublime Text**, **Atom**).

Example: Open the browser console and type the following command:

console.log("Welcome to JavaScript!");

1.6 JavaScript Versions (ECMAScript)

JavaScript follows the ECMAScript standard (ES) with periodic updates that introduce new features.

Key versions include:

  • ES5 (2009): Introduced JSON support, strict mode, and new array methods.
  • ES6 (2015): Introduced let, const, arrow functions, template literals, and classes.
  • ES7 - ES13: Brought in async/await, optional chaining, and other modern features.

Staying up-to-date with ECMAScript updates helps developers write modern, efficient JavaScript code.

1.7 Summary

In this module, you learned:

  • JavaScript is a dynamic programming language used for web interactivity.
  • It works alongside HTML and CSS to create modern web applications.
  • There are different ways to include JavaScript in a webpage.
  • JavaScript runs on browsers using JavaScript engines.
  • Modern JavaScript follows ECMAScript standards.
