Interactive Web Page Practice

In this hands-on practice, you will create a simple interactive web page that includes buttons, text inputs, and dynamic content updates using JavaScript.

6.1 Creating a Simple Web Page

Step 1: Create an HTML structure

<div id="container">
    <h2>Welcome to My Interactive Web Page!</h2>
    <button onclick="showAlert()">Click Me</button>
    <p id="dynamicText">Hello, World!</p>
</div>

Step 2: Style the Page with CSS

#container {
    background-color: #282828;
    color: #fff;
    padding: 20px;
    border-radius: 8px;
}
button {
    background-color: #66fcf1;
    color: #1f1f1f;
    padding: 10px;
    border: none;
    border-radius: 50px;
}
button:hover {
    background-color: #45a29e;
}

Step 3: Use JavaScript for Interactivity

function showAlert() {
    alert("Button clicked!");
}
