The Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface that allows JavaScript to interact with and manipulate HTML documents. It represents a document as a tree of objects and nodes that can be dynamically modified.

4.1 What is the DOM?

The DOM allows developers to interact with a webpage's structure programmatically. It represents the document's structure as a tree of nodes, where each node is an element in the document.

  • DOM Tree: Every element in an HTML document becomes a node in the DOM tree.
  • Manipulating the DOM: JavaScript can add, remove, or modify DOM elements.

4.2 Selecting Elements

Before you can manipulate an element, you need to select it from the DOM. Here are a few ways to select elements:

1. Using getElementById:

Use getElementById to select an element by its unique ID.

let element = document.getElementById("header");

2. Using getElementsByClassName:

Use getElementsByClassName to select elements by their class name. This returns a collection of elements.

let elements = document.getElementsByClassName("item");

3. Using querySelector:

Use querySelector to select the first element that matches a CSS selector.

let element = document.querySelector(".menu");

4.3 Manipulating Elements

Once you've selected an element, you can manipulate it in various ways such as changing its content, styles, or adding/removing it from the document.

1. Changing Content:

Use textContent or innerHTML to change the content of an element.

let header = document.getElementById("header");
header.textContent = "Welcome!";

2. Changing Styles:

Use JavaScript to change the style of an element directly. For example, you can change its color, font size, and more.

header.style.color = "blue";
header.style.fontSize = "24px";
