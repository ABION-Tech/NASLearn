Hands-On Practice: Building a Small Project Using React

4.1 Setting Up the Project

  • Install Node.js and npm: Ensure you have Node.js and npm installed on your machine.
  • Create a React App: Use create-react-app to quickly set up a new React project. 
npx create-react-app my-react-app
cd my-react-app
npm start

4.2 Developing the Project

Students will create a simple React application that allows users to input their name and see a personalized greeting. The project will involve:

  • Creating Components: Break down the UI into reusable components.
  • Managing State: Use React's state to manage user input and display dynamic content.
  • Handling Events: Implement event handlers for user interactions.

4.3 Styling the Application

Use CSS to style the React components.

Explore using a CSS framework like Bootstrap with React.

4.4 Deploying the Project

Deploy the React application using a platform like GitHub Pages, Vercel, or Netlify.

Tableau Project

Tableau Project

Objective

Apply Tableau skills learned throughout the week to create a business intelligence dashboard using real-world data. The project involves connecting to data sources, designing visualizations, building interactive dashboards, and presenting insights.

Dataset

Use a provided dataset (e.g., sales performance, customer demographics, marketing campaign data) for the project.

Tasks

  • Data Connection: Connect Tableau to the dataset and perform data preprocessing if necessary.
  • Visualization Design: Create multiple visualizations (e.g., charts, graphs, maps) to explore different aspects of the data.
  • Dashboard Creation: Build a cohesive dashboard with interactive elements (e.g., filters, parameters) to convey insights effectively.
  • Presentation: Prepare a presentation to showcase the Tableau dashboard, highlighting key findings and actionable insights.

Report Writing

Write a report documenting the dashboard design process, data analysis techniques applied, and recommendations based on the insights gained.
