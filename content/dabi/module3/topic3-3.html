SQL Data Manipulation

INSERT Statement

The INSERT statement is used to add new rows to a table. 

INSERT INTO customers (customer_name, contact_name, country)
VALUES ('John Doe', 'John', 'USA');

UPDATE Statement

The UPDATE statement modifies existing rows in a table. 

UPDATE customers
SET contact_name = 'Jane'
WHERE customer_name = 'John Doe';

DELETE Statement

The DELETE statement removes rows from a table. 

DELETE FROM customers
WHERE customer_name = 'John Doe';

Data Integrity

Maintaining data integrity involves ensuring the accuracy and consistency of data within a database. Key concepts include:

  • Primary Keys: Unique identifiers for each row in a table.
  • Foreign Keys: Columns that create a relationship between tables by referencing primary keys.
  • Constraints: Rules applied to columns to enforce data integrity (e.g., NOT NULL, UNIQUE, CHECK).

Transactions

Transactions ensure that a series of SQL operations are executed as a single unit, maintaining data integrity. If any operation fails, the entire transaction is rolled back. 

BEGIN TRANSACTION;
UPDATE accounts SET balance = balance - 100 WHERE account_id = 1;
UPDATE accounts SET balance = balance + 100 WHERE account_id = 2;
COMMIT;
